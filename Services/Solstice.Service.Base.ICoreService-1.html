Interface ICoreService<T>

Namespace
Solstice.Service.Base
Assembly
Solstice.Applications.dll

This interface describes an abstraction for a data manipulation service on a given type of RadiantModel. It includes methods for CRUD operations, queries for specific entities, counting entities, starting transactions, and more.

public interface ICoreService<T> where T : class
